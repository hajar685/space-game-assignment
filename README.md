# Space Invaders Game

This project is a simple browser-based Space Invaders style game built using:

- HTML
- CSS
- JavaScript
- HTML5 Canvas

The player controls a spaceship and must shoot incoming enemies while avoiding enemy attacks.

## Controls
- Left Arrow → Move left
- Right Arrow → Move right
- Spacebar → Shoot

## Features
- Enemy movement
- Player shooting
- Enemy shooting
- Score system
- Lives system
- Game over condition
- Restart button
